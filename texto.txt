SPRINt 04 

Front-end Design 
Deve utilizar o mesmo projeto com base no Front-End da Sprint anterior. 
LEmbre-se que essa base será integrada ao React na disciplina de JavaScript 
Requisitos: 
- Integração completa com React 
- Utilização de Grid Layout, Flexbox e Bootstrap de forma correta (10 pontos);
- Ultilização de HTMl semântico (10 pontos);
- Utilização de Medias Queries (10 pontos); 
- Responsividade Completa do projeto (Desktop - Tablet - Mobile) (40 pontos);
Entrega: link do repositório (público) no Github. 

Web Development 
- Projeto criado em React utilizando estrutura de componentes e tratamento de estilização com Bootstrap (20 pontos);
- Projeto em React precisa consumir de uma API utilizando recursos de json(Local), revisão do DOM e criação de eventos 
    nas partes do projeto, estilização com sass (60 pontos);
- Estilizar o projeto com Sass e Bootstrap, dentro dos componentes criados utilizar um pelo menos para consumir de uma API. (20 pontos);
Regras: 
- Pode utilizar a documentação do React como guia; 
- O projeto deve ser criado dentro das normas W3C; 
- O projeto precisa ter criação própria utilizando os conhecimentos adquiridos; 
- A estrutura deve ser semântica; 
- Versionamento do projeto usando organização no Github; 
Entrega: O formato da entrega deve ser um arquivo compactado sem os módulos;.
